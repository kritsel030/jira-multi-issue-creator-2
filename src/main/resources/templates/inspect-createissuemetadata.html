Inspect - createissuemetadata

Displays the raw result of the GET /rest/api/3/issue/createmeta Jira API endpoint.
See https://developer.atlassian.com/cloud/jira/platform/rest/v3/api-group-issues/#api-rest-api-3-issue-createmeta-get for more info.

Note that you can only retrieve this metadata for projects you are allowed to create issues in. If there's no result, the project either not exists, or you do not have the 'create issue' permissions for it.
